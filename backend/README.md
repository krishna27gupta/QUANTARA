# Quantara Backend Service

FastAPI Python 3.13 backend service for stock prediction and analytics.

## Developer Setup

1. Make sure Python 3.13 and Poetry are installed.
2. Run `poetry install` in this directory to install dependencies.
3. Configure your local `.env` file based on `.env.example`.
4. Run `poetry run uvicorn app.main:app --reload` to start development server.
